# Security Policy

## Supported Use

This project is intended for local, personal patching of an already installed
Workshop mod DLL. Do not run patchers or scripts downloaded from an untrusted
source.

## Reporting

Please report issues with:

- the command you ran,
- your game and Workshop paths,
- the patcher output,
- whether Cities: Skylines was running.

Do not attach proprietary game files or Workshop DLLs to public issues.
