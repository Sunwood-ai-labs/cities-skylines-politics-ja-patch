# Notice

This repository contains a local binary patcher for the Cities: Skylines 1
Workshop mod "Politics & Elections".

It does not include, redistribute, or claim ownership of the original mod,
Cities: Skylines, Steam Workshop content, or the Mono.Cecil DLL bundled with
the game. Users must already have the game and the Workshop mod installed.

Back up your save and Workshop files before patching. The helper script creates
a timestamped DLL backup by default.
